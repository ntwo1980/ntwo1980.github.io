Termux的安装和连接

发表于 2017-09-22
  1. 安装完成Termux后，运行apt update && apt upgrade和apt install openssh。
  2. 运行termux-setup-storage
  3. 在PC上将public key拷贝到手机DCIM目录。
  4. 创建.ssh目录，执行mkdir .ssh
  5. 执行cat storage/dcim/id_rsa.pub >> .ssh/authorized_keys
  6. 执行sshd,启动ssh服务。
  7. 执行whoami，查看当前用户名。
  8. 在PC上连接Termuxssh -p 8022 u0_a130@10.10.10.10
