衡量公司各项能力的指标

发表于 2017-09-13

一般而言，投资者关注的财务指标衡量了公司在以下七个方面的表现：

盈利能力

财务比率 计算公式 数据来源 含义
资本回报率或净资产收益率（Return-on-equity, ROE） 税后净利润 / 平均所有者权益 利润表和资产负债表 衡量了公司股东投资的资产所实现的回报。资本回报率越高，说明公司对资本金的利用与管理越有效，带给股东的保障和回报越多。但是，杠杆率过高也可能导致资本回报率仅仅是看起来很高，因此该指标须结合资产回报率和资本结构进行判断。因为优先股的性质类似债权，所以资本回报率通常衡量的是普通股股权部分的盈利能力。如果公司也发行了优先股，一般会从净利润中减去优先股股利，从净资产中减去优先股股权再进行计算。
资产回报率（return-on-asset，ROA） 税后净利润 / 平均总资产 利润表和资产负债表 综合衡量了公司利用资产获得回报的效率。一般而言，资产回报率越高的公司其盈利能力和经营管理水平越高；但是，轻资产行业（高科技行业、服务业等）公司的资产回报率普遍高于重资产行业（固定资产比重较高）。有时，也有以使用营业利润代替净利润进行计算。
销售毛利率 （销售收入 - 销售成本） / 销售收入 利润表 反映了公司对原材料的利用效率和产品的初始获利能力。该指标较高的公司，一方面可能与供应商的议价能力较强或者享受规模经济效应，从而可以压低产品成本；另一方面可能有着很高的品牌效应，因而定价较高。竞争越激烈的行业，销售毛利率一般越低；销售毛利率越高的公司，在行业中的竞争优势越强。
主营业务利润率 （销售毛利 - 各种经营费用） / 销售收入 利润表 剔除了非销售利润因素，直接反映了企业经营活动的盈利能力。因为企业的经营管理很大一部分在于控制种类营业费用，所以该指标的变化趋势在很大程度上反映了企业的经营管理能力。
净利率 净利润 / 销售收入 利润表 最直接、简单地衡量了企业将销售额转化为股东利润的能力。

短期偿债能力（流动性水平）

财务比率 计算公式 数据来源 含义
流动比率 流动资产 / 流动负债 资产负债表 衡量了作为流动负债偿还保障的流动资产水平。流动比率越高，一方面意味着债权人面临的违约风险越小，另一方面说明了企业运营资金越充足。但是，过高的流动比率可能暗示着企业存货积累过多、应收账款难以回收或者现金管理出现了问题，这些都是企业盈利能力低下的隐患。
速动比率 （流动资产 - 存货 - 其他流动资产) / 流动负债 资产负债表 更加保守地衡量真正能够迅速变现、以偿还短期负债的那部分流动资产水平。由于存货和预付费用等流动性资产变现难度大、跨期长并且会有所折价，因此该比率从流动资产中剔除以存货为主的其他流动资产。速动比率偏低的企业面临较大的流动性风险。如果速动比率与流动比率之间的差异很大，则意味着企业的流动性主要靠大量存货囤积形成。
现金比率 （现金及现金等价物 + 短期可交易证券) / 流动负债 资产负债表 作为最保守、严苛的流动性比率，衡量了能迅速用于偿还短期债务的现在持有量。但对于企业而言，持有过多现金也是一种资源浪费，因此现金比率的适用性十分有限。

清偿能力

财务比率 计算公式 数据来源 含义
负债比率（资产负债率） 总负债 / 总资产 资产负债表 衡量企业所运用的财务杠杆。负债比率越高，说明公司总资产中来自债权人的资金真多；杠杆越高，公司相应承担的风险越大。财务杠杆是把“双刃剑”，它会同时放大企业的盈利和损失。需要注意的是，负债比率中包含了一部分由经营业务产生的必要债务（应付账款）。
产权比率 总负债 / （普通股）所有者权益 资产负债表 衡量所有者权益对于投资人的保障程度，反映了企业财务结构的稳健性。产权比率越小，代表公司的账务杠杆越低，因而承担的风险也越小。有时投资者会用一个更加保守的产权比率（将分母替换为有形净资产，即所有者权益 - 无形资产），此时一些依靠大量并购积累声誉的企业的有形净资产可能为负值。
利息保障倍数 息税前利润（EBIT）/ 利息费用 利润表 衡量企业依靠盈利所得偿付债务的能力。利息保障倍数越低，企业对当前债务水平的承担能力越弱，也会严重影响企业的盈利能力。一般而言，该指标低于1.5的企业，其清偿能力存在一定问题。判断一家企业清偿能力的持续性，应注意分析其在一个经济周期内利息保障倍数的变化。

营运能力

财务比率 计算公式 数据来源 含义
存货周转率 销售成本 / 平均存货 利润表和资产负债表 反映企业存货周转速度和销货能力。存货周转率高的公司，存货出售得更快，存货管理管理效率更高，占用的资源更少，积压的风险也更小。但过高的存货周转率也可能意味着存货过少，难以应对增长的市场需求。
应收账款周转率 销售收入 / 平均应收账款 利润表和资产负债表 反映企业应收账款变现的速度和效率。应收账款越高，公司的收账能力越强，因而流动性越强，坏帐率一般也越低。但是，过高的应收账款周转率也暗示了企业赊销规定可能过于严苛，导致其失去了一些销售机会。
总资产周转率 销售收入 / 平均总资产 利润表和资产负债表 衡量企业利用资产产生收入的效率。总资产周转率低的企业，其利用资产的效率低或者在资本密集的环境下运营。当然，这也可能与企业选择资本密集型劳动力密集型的策略有关。
固定资产周转率 销售收入 / 平均固定资产净值 利润表和资产负债表 衡量企业的固定资产产生收入方面的生产力，反映企业对固定资产的管理效率。对于重资产行业的企业，这项指标发挥的作用更大。该指标越高，企业的固定资产利用率越高，投资效率越高、营运能力越强。但是，过高的固定资产周转率也能暗示企业的固定资产扩张受到一定的限制，未来的规模增长较难实现。

成长能力

财务比率 计算公式 数据来源 含义
销售收入同比增长率 本年度销售收入 / 上一年度销售收入 - 1 利润表 衡量公司一个会计年度内销售收入的变化程度。一般而言，销售收入增长率越高的公司，其市场占有率扩张得越快，未来成长也更乐观。
净利润增长率 本年度净利润 / 上一年度净利润 - 1 利润表 直接衡量公司当期净净利润与上一年度相比的增长幅度，净利润增长率越高的公司，其盈利能力越强。
总资产增长率 本年年末总资产 / 上一年年末总资产 - 1 资产负债表 衡量公司在一个会计年度内资产规模的增长状况。资产是公司取得收入的资源和偿还债务的保障，发展性强的公司一般会保持稳定的资产增长率。
固定资产占比 本年年末固定资产 / 本年年末总资产 资产负债表 衡量公司是否存在固定资产导致的资金闲置现象。从资产运用角度而言，固定资产占比越低的公司，其资产利用效率越高，但不同行业在该比率上可能存在较大的差异。

现金流情况

财务比率 计算公式 数据来源 含义
单位销售现金净流入 经营活动产生的净现金流 / 销售收入 现金流量表和利润表 衡量企业将销售额转化为现金的能力。能够维持较高单位销售现金流入比率的公司，其获取现金的能力更强，流动性风险和财务造假可能性更小。
债务保障率 经营活动产生的净现金流 / (流动负债 + 长期负债) 现金流量表和资产负债表 衡量企业现金流对债务偿还的充足程度，更真实和保守地反映了企业的清偿能力。债务保障率越高的企业，其违约风险越小，发展潜力越大。
自由现金流与经营活动净现金流比值 (经营活动净现金流 - 资本支出) / 经营活动净现金流 现金流量表 自由现金流是企业真正能用于扩展业务、进行并购和投资等其他非主营型业务的资金来源。因此，自由现金流占经营活动产生的净现金流比例越高的企业，其财务实力越雄厚，投资质量越高。

市场表现

财务比率 计算公式 数据来源 含义
每股收益（earnings-per-share, EPS） 归属于普通股股东的净利润 / 对外发行的普通股股数 利润表和市场数据 反映普通股股东投资的获利水平。一般可以从数据库中获利两种每股收益，其中稀释每股收益是将企业发行的可转债等潜在股权工具也计入普通股股数得到的每股收益，是一种更保守的算法。
市盈率（price-to-earnings, P/E） 普通股每股市价 / 每股收益 利润表和市场数据 在相对估值模型中最常用的市场倍数之一。市盈率高的公司，一般在未来短期内会有更高的盈利增长率；但同时，市盈率过高的公司，其被市场高估的可能性较大。
市净率（price-to-book, P/B） 普通股每股市价 / 每股账面价值 资产负债表和市场数据 衡量新股东为了获利公司净资产所付出的投资额或者经理人为现有股东创建财富的能力，给投资者提供了市场对公司估值与公司清算时股东所能获得的价值的差异。市净率较低的公司存在股价被低估的可能性，但也有可能是由投资滞后所导致。市净率更适用于对资本密集型或金融行业的公司进行估值。
市销率（price-to-sales, P/S） 普通股每股市价 / 每股销售收入 利润表和市场数据 类似市盈率，衡量公司获得单位销售收入所需的股东投资额。销售收入受会计准则的影响更小，因此市销率是一个比市盈率更可靠的估值乘数。市销率偏低的公司被低估的可能性更大。

收入净利率因子

发表于 2017-09-11
